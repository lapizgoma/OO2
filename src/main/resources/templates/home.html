Sistema de Tickets UNLa

Una solución eficiente para gestionar tickets de soporte en la Universidad Nacional de Lanús.

Características

Creación de Tickets

Crea tickets de soporte de manera rápida y sencilla, con seguimiento en tiempo real.

Gestión por Roles

Diferentes permisos para usuarios, empleados y administradores, optimizando la gestión.

Historial y Reportes

Accede al historial de tickets y genera reportes para un mejor seguimiento.

Tecnologías Utilizadas

Spring Boot

Backend robusto y escalable para gestionar la lógica del sistema.

Thymeleaf

Motor de plantillas para una integración fluida con el backend.

Bootstrap

Diseño moderno y responsive para una experiencia de usuario óptima.

Java

Lenguaje principal para el desarrollo del backend.
